# pangashivoter

A voter portal app with admin login, Supabase-backed auth, and voter management.

## Run Locally

1. Install dependencies:
   `npm install`
2. Start the app:
   `npm run dev`
3. Open http://localhost:3000

## Notes
- Environment values should be kept in [.env.local](.env.local)
- Demo admin credentials are disabled; use a real authenticated admin account
